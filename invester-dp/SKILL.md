# 三步投资法 · 决策导航助手

## Overview

**你不是不会用工具，你是不确定现在该用哪个工具。**

这个 Skill 就是一个**投资决策导航**。你只需描述你当下的状态，它就告诉你：

1. 你现在处在什么阶段？
2. 应该用三个 Skill 中的哪一个？
3. 下一步需要做什么？
4. 输出形式是什么？

---

## 三句话记住三个 Skill

| Skill | 一句话 | 解决什么问题 |
|:---|:---|:---|
| **hotspot-chain-hunter** | **"热点在哪里？产业链怎么拆？"** | 不知道市场在炒什么，跟哪条线 |
| **six-dimension-hunter** | **"这家公司值得买吗？"** | 已经看到某家公司，想知道能不能长期持有 |
| **thesis-tracker** | **"我买的这家现在怎么处理？"** | 已经买了，该加仓/减仓/清仓/持有 |


## 常见误区

| 误区 | 你可能会问 | 正确做法 |
|:---|:---|:---|
| **用错 Skill** | "我有持仓了，帮我用六维捕手重新评一下" | 应该用 thesis-tracker 做持仓评估，不是重新评分 |
| **跳步** | "帮我用热点深挖选XX公司" | 先跑 hotspot-chain-hunter 确定方向，再跑 six-dimension-hunter 评分 |
| **不做建档** | "我买了直接监控就行了吧" | 没有投资档案的持仓，遇到波动会不知道是否该卖 |
| **不做复盘** | "卖了就卖了，懒得复盘" | 不复盘等于"用一个亏钱的经验换了一个更贵的亏钱经验" |
| **频繁换股** | "这个涨了换那个" | 持仓未满3个月不轻易换，除非核心假设被推翻 |

## 三个 Skill 的依赖关系

```
hotspot-chain-hunter
    │ (输出候选标的)
    ▼
six-dimension-hunter
    │ (输出评分+买入区间)
    ▼
thesis-tracker
    │ (建仓档案 → 监控 → 加减仓 → 退出复盘)
    ▼
经验反哺 ←────────── 回到热点扫描，优化方向判断
```

**建议的使用节奏**：
- **每周一**：用 `hotspot-chain-hunter` 跑一次热点扫描（日常盘前10分钟）
- **选中标的**：用 `six-dimension-hunter` 跑一次评分（周末30分钟）
- **已建仓标的**：按 thesis-tracker 的节奏复盘
- **每月第一个周末**：thesis-tracker 场景3（组合体检30分钟）
