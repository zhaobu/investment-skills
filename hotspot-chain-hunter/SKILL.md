# 热点产业链深挖选股器 · Hotspot Chain Hunter

## Overview

**核心思想**：不做"纯等风来"（资金效率低），也不做"纯追情绪"（基本面没底）。
而是：**先跟市场热钱走，确定当前资金聚集的板块方向；再对该方向做全套产业链深挖和六维筛选，选出热钱方向里基本面最硬、估值最合理的标的。**

### 典型触发场景
- "当前热点是什么，怎么选股？"
- "今天资金在炒什么方向，帮我挑里面的好公司"
- "帮我分析XX板块的产业链，每个环节选最好的标的"
- "看看现在市场在交易什么主线，从中选股"
- "最近XX概念很火，帮我做全产业链分析+选股"
- 任何提及"热点/资金流向/主线/板块"且需要选股结论的请求

### 不适用场景
- 纯大盘分析、纯技术面分析、纯宏观判断（不涉及具体选股）
- 已有明确目标个股的深度研究（直接用 `six-dimension-hunter`）
- 港股/美股为主体的需求（仅支持A股，如需跨境分析需显式标注）

## 核心流程

```
Step 0: 市场环境判断 → 确定当前适合进攻/精选/控制/观望
  │
Step 1: 市场热点扫描 → 锁定当前资金聚集方向（Top 3-5）
  │
Step 1.5: 热点持续性验证 → 过滤一日脉冲假热点
  │
Step 2: 热点含金量评估 → 剔除纯情绪炒作方向（含生命周期位置判断）
  │
Step 3: 确定1-2个高含金量方向 → 拆解该方向的完整产业链
  │
Step 4: 产业链子板块逐一六维评分 → 每个子板块精选Top 1-2标的
  │
Step 5: 排除分析 → 说明未入选标的的短板
  │
Step 6: 输出完整报告（含潜伏池+触发条件+仓位建议）
```

## Step 0: 市场环境前置判断

**在扫描热点之前，先判断当前市场大环境适合什么打法。** 同样的热点，强市里可以重仓追，弱市里必须轻仓快进快出。

### 市场环境四象限

| 市场状态 | 判断标准 | 适合策略 | 仓位建议 |
|:---|:---|:---|:---:|
| 🟢 **强势市场** | 指数多头排列+涨跌比>1.5+成交额>8000亿+情绪热烈 | 进攻（重仓做多） | 7-9成仓 |
| 🟡 **震荡市场** | 指数无趋势+涨跌比1:1+成交额5000-8000亿+轮动为主 | 精选（聚焦核心方向） | 4-6成仓 |
| 🔴 **弱势市场** | 指数空头+跌多涨少+成交额<5000亿+情绪冰点 | 控制（轻仓或空仓） | 0-3成仓 |
| ⚫ **退潮市场** | 指数连续破位+跌停大幅增加+成交额萎缩 | 观望（不操作） | 0成仓 |

### 数据获取方式

- 指数走势：`neodata "上证指数 2026年 走势 均线"` + `westock-data kline sh000001 --period day --limit 20`
- 成交额：`neodata "A股 当日成交额"`
- 涨跌家数：`neodata "A股 涨跌家数 今日"`
- 情绪判断：`westock-data hot stock` 查看连板高度 + 涨停数量

### 判断输出

```
📊 市场环境：[强势/震荡/弱势/退潮]
建议策略：[进攻/精选/控制/观望]
建议仓位上限：[X成仓]
```

> **关键规则**：在🔴弱势或⚫退潮市场下，默认跳过产业链深挖（Step 3-6）。例外：如果该方向属于防御性板块（医药、必选消费、公用事业），或在弱市中资金持续逆势流入，可适当降低仓位参与，但仍不进行全产业链拆解。

### ⚠️ 边界情况处理

| 特殊情况 | 处理规则 |
|:---|:---|
| **弱市但方向是防御性板块**（医药/必选消费/公用事业） | 允许做Step 3-6，但仓位上限控制在3成以内 |
| **弱市但该方向存在政策强行催化**（如突发重磅政策） | 允许做Step 3-6，但标注"环境风险"，仓位上限5成 |
| **震荡市但多个方向同时触发** | 选持续性最好的1个方向深挖，其余进入"监控名单" |
| **方向横跨上下游且公司同时受益**（如"AI+医药"） | 按主要收入来源（>60%）归入对应产业链模板 |
| **同板块多只标的六维评分接近（分差<10分）** | 优先选择PE更低的那只，其次选择现金流更好的那只 |

## Step 1: 市场热点扫描

动用所有可获取的市场数据，识别当前A股市场正在交易的**核心热点方向**。

### 数据来源清单

| 数据类型 | 获取方式 | 具体内容 |
|:---|:---|:---|
| 板块涨跌幅排名 | `neodata "A股 板块涨幅 排名 近5日"` + `neodata "板块 涨幅 Top 20"` | 近5日/20日板块涨幅Top 20 |
| 成交量/成交额异动 | `neodata "板块 资金流向 成交额"` | 近5日成交额放大最显著的板块 |
| 北向资金流向 | `neodata "北向资金 板块 流向"` | 近1周/1月北向净买入Top 10 |
| 主力净流入排名 | `neodata "主力资金 板块 净流入"` | 当日/近5日主力净流入Top 10 |
| 政策催化剂 | `WebSearch "A股 最新政策 利好 板块"` | 近期政策催化 |
| 涨停家数/板块分布 | `neodata "涨停 板块 排名"` | 涨停股最多的板块 |
| 机构仓位变化 | `neodata "公募基金 板块 持仓 变动"` | 机构在加仓什么方向 |
| 概念热搜排行 | `neodata "市场 热点概念"` | 当前最热的概念主题 |

### 输出要求

**限取 Top 5 个热点方向**，每个方向给出以下信息：

```
| 排名 | 方向名称 | 当日涨跌幅 | 近1月涨幅 | YTD涨幅 | 核心驱动 | 热度评级 | 数据源 |
|:---|:---|:---:|:---:|:---:|:---|:---:|:---|
```

**热度评级标准：**
- 🔴高热：近1月涨幅>30% 或 PE中位数>80 — 追高风险大，仅短线
- 🟡适中：近1月涨幅15-30% 且 PE中位数30-60 — 可做中期
- 🟢冷门：近1月涨幅<15% 且 PE中位数<30 — 可能正在启动，性价比最佳

## Step 1.5: 热点持续性验证

**防止你被"一日脉冲"骗进去做全套分析。** 单日放量可能是假消息驱动，第二天就熄火了，不值得花2小时拆产业链。

### 持续性四维验证

| 验证维度 | 持续性强的信号 | 一日游信号 |
|:---|:---|:---|
| ①成交额持续性 | 该方向成交额连续3日>5日均量 | 仅单日放量，次日缩量>30% |
| ②中军股表现 | 至少1只中军股(成交额>30亿/日)持续上涨 | 全是小盘股涨停，大市值股不动 |
| ③催化验证 | 有明确的产业逻辑/政策文件/订单落地 | 仅有朦胧消息或网络传言 |
| ④资金无出逃 | 龙头未出现涨停板大额净流出 | 龙头涨停但主力净流出超5亿 |

### 验证流程

```
对各热点方向逐一检查：
├─ 满足全部4条 → ✅ 真主线，正式进入后续分析
├─ 满足3条 → ⚠️ 准主线，分析但降低优先级
├─ 满足1-2条 → ❌ 疑似一日游，跳过深挖，仅监控
└─ 0条 → ❌ 纯脉冲，直接放弃
```

### 数据获取方式

| 验证维度 | 数据获取 | 说明 |
|:---|:---|:---|
| ①成交额持续性 | `westock-data sector <板块代码>` 取近5日成交额 + 手动对比连续趋势 | 看是否连续3日>5日均量 |
| ②中军股表现 | `westock-data quote <大市值龙头代码>` 查成交额>30亿 | 看大市值股是否动 |
| ③催化验证 | `WebSearch "方向名 最新政策 订单 2026"` + `neodata "方向名 最新消息"` | 验证消息真伪 |
| ④资金无出逃 | `westock-data asfund <涨停股代码>` 查主力净流向 | 看涨停是否伴有主力净流出 |

## Step 2: 热点含金量评估

对每个热点方向，判断其热度背后是否有**基本面支撑**，以及该热点处于**题材生命周期的哪个阶段**。

### 五维含金量判断矩阵

| 判断维度 | ✅ 高含金量(可信) | ❌ 低含金量(炒作) |
|:---|:---|:---|
| ①盈利支撑 | 该方向多数公司盈利正增长 | 多数亏损或微利 |
| ②机构认同 | 公募/北向持续加仓 | 游资主导，机构不碰 |
| ③政策落地 | 已有明确政策文件或订单落地 | 仅"文件"无细则无订单 |
| ④产业趋势 | 产业正处在爆发初期(渗透率<20%) | 产业在成熟晚期或纯概念阶段 |
| ⑤估值安全 | PE_fwd<50, 有增长消化空间<br>（高增长赛道如AI/算力/创新药可放宽至<80） | PE_fwd>100, 纯靠情绪 |

### 生命周期位置判断（新增）

**同一个方向，在生命周期不同位置，操作策略完全不同。**

| 阶段 | 识别标准 | 对应策略 | 是否深挖 |
|:---|:---|:---|:---:|
| 🟢 **萌芽期** | 机构刚覆盖+涨幅<20%+PE合理+散户讨论低 | **重仓买入**，深挖选股 | ✅ |
| 🟡 **主升期** | 涨幅20-80%+共识形成中+成交量温和放大 | **核心仓位**，严格止盈 | ✅ |
| 🔴 **高潮期** | 涨幅>150%+机构全面覆盖+PE透支+散户狂热 | **不参与或极轻仓短线** | ❌ 仅短线 |
| ⚫ **退潮期** | 龙头开始跌停+板块轮动+成交量萎缩 | **清仓不碰** | ❌ 跳过 |

**快速判断生命周期的方法**：
1. 看YTD涨幅：<20%=萌芽, 20-80%=主升, >150%=高潮, 高位回落>30%=退潮
2. 看机构覆盖：`westock consensus` 的 `institutionCnt`，机构数从少变多=主升中
3. 看散户讨论热度：`neodata "热点概念 搜索 热度"`，讨论火爆=高潮末期

### 判断结论

| 结果 | 含义 | 行动 |
|:---|:---|:---|
| ✅ **高含金量 + 未到高潮期** | 进入产业链深挖 | 进入 Step 3 |
| ⚠️ **高含金量但进入高潮期** | 仅短线，不做深挖 | 进入"短线监控名单" |
| ⚠️ **含金量中等但热度强** | 仅观望，不做深挖 | 进入"监控名单" |
| ❌ **纯情绪炒作 / 退潮期** | 直接排除 | 不分析 |

## Step 3: 产业链拆解

从Step 2筛选出的✅高含金量方向中，选择**热度评级最好且基本面最强的1-2个方向**，进行完整的产业链拆解。

### 产业链拆解方法

对每个选定方向，先判断该方向的**产业链类型**，再套用对应的拆解模板：

#### 四种产业链拆解模板

| 类型 | 适用方向 | 拆解模板 | 实战案例 |
|:---|:---|:---|:---|
| **A型：制造业链** | 储能、新能源车、光伏、PCB、化工 | 上游(原材料/设备)→中游(制造/加工)→下游(终端应用) | 锂电池：锂矿→正极/负极/电解液→电芯→电池包→整车 |
| **B型：TMT链** | AI、半导体、软件、5G、光通信 | 上游(底层硬件/IP)→中游(芯片/模组/平台)→下游(应用/服务) | AI：算力芯片→服务器→模型层→应用层 |
| **C型：消费链** | 白酒、医美、食品、家电 | 上游(原料/包材)→中游(品牌/生产)→下游(渠道/零售) | 白酒：粮食/包材→酒企→经销商→终端 |
| **D型：服务链** | 金融科技、医疗、教育 | 上游(基础设施)→中游(平台/服务商)→下游(终端用户) | AI医疗：医疗数据→AI诊断平台→医院/患者 |

#### 选择模板规则
1. 根据方向所属行业，选择对应的模板
2. 如果方向横跨多个类型（如"储能"既有制造业属性也有能源服务属性），以**价值量占比最高的环节为准**
3. 对于新兴/跨界热点方向，使用最接近的模板类型，并根据实际情况调整子板块划分

**每个子板块需列出：**
- 子板块名称
- 核心业务描述（一句话）
- 在该方向中的价值量占比（估算）
- 该子板块的竞争格局（集中/分散/寡头）
- A股主要上市公司清单（代码+名称）

> **数据获取方式**：`neodata` 查询产业链结构 + `westock-data sector --search` 找板块成分股 + WebSearch补充

### 产业链优先级排序

对拆解后的所有子板块，按以下标准决定**分析优先级**：
1. **价值量高**的子板块优先（毛利率高、不可替代性强）
2. **国产替代空间大**的子板块优先（技术壁垒高、国内企业少）
3. **直接受益AI/数字化转型**的子板块优先（增量需求明确）

### ⚠️ 方向质量检查（新增）

在进入六维评分前，先对整个方向做一次**"有无好标的"预检**。避免拆了两小时后发现该方向里全是垃圾股。

| 检查项 | 判断标准 | 结果 |
|:---|:---|:---|
| 板块总市值 | 该方向A股上市公司的中位数市值  | >100亿→正常 / <30亿→偏小市值，注意流动性 |
| 是否有明确龙头 | 至少1家公司营收>100亿且净利>10亿 | 有→正常 / 无→该方向"大行业小公司"，降低预期 |
| 机构覆盖密度 | 核心标的的研报机构数（consensus中的institutionCnt） | >10家→正常 / <3家→该方向关注度极低 |
| 同质化程度 | 核心标的是否有明显差异化 | 有壁垒差异化→可评 / 高度同质化→难选龙头 |

**结论**：
- ✅ 全部通过 → 正常进入Step 4评分
- ⚠️ 部分未通过（如市值偏小但龙头明确） → 降低仓位预期，仍可评分
- ❌ 无龙头+无机构覆盖+高度同质化 → 该方向虽然逻辑对但A股无合适标的，输出"建议通过ETF参与"

## Step 4: 子板块六维评分筛选

对每个高优先级的子板块，执行完整的六维捕手评分。

### 4.1 排雷预筛选

进入评分前，先对每个子板块内的标的执行排雷：

| 淘汰类型 | 判断标准 | 检查方式 |
|:---|:---|:---|
| **主题相关性不足** | 核心业务与主题方向无关，或收入占比<10% | `neodata` 主营构成 + `westock profile` |
| **估值泡沫** | PE_fwd>150 或 亏损且无改善预期 | `westock quote` |
| **炒作过度** | YTD涨幅>150% 且 净利增速<50% | `westock quote` + `consensus` |
| **纸面利润** | 经营现金流/净利润<50%（连续3年） | `westock finance` |
| **强周期陷阱** | PE极低但行业处于盈利高点，不可持续 | 毛利率趋势+供需周期判断 |

### 4.2 六维逐项评分

沿用标准六维捕手框架：

#### 维度①：稀缺性（★★★★）
| 分数 | 标准 |
|:---|:---|
| 7-10分 | 全球仅1-5家能做，有技术/认证/资源壁垒 |
| 4-6分 | 同类5-10家，有一定壁垒 |
| 1-3分 | 大宗同质化，谁都能做 |

**数据获取**：`neodata` 查询行业研报竞争者数量 + `westock sector` 统计成分股

> **淘汰线**：<4分直接淘汰

#### 维度②：龙头地位（★★★★）
| 分数 | 标准 |
|:---|:---|
| 7-10分 | 全球#1或国内#1，市占率>20% |
| 4-6分 | 国内Top 3但非龙头 |
| 1-3分 | 无龙头地位 |

**数据获取**：`westock report` 研报标题判断 + `neodata` 查询市占率

> **淘汰线**：<4分直接淘汰

#### 维度③：3-5年确定性（★★★★）
| 分数 | 标准 |
|:---|:---|
| 7-10分 | 不可绕过的物理/商业瓶颈，或下游需求确定性强 |
| 4-6分 | 有增长但存在替代路径 |
| 1-3分 | 概念性/周期性强 |

**数据获取**：`westock consensus` 未来3年净利CAGR + 行业需求驱动判断

> **淘汰线**：<4分直接淘汰

#### 维度④：估值未透支（★★★★）
| 分数 | 标准 |
|:---|:---|
| 7-10分 | PE_fwd<30 且 2028E PE<20 |
| 4-6分 | PE_fwd 50-80, 尚可接受 |
| 1-3分 | PE_fwd>80 或 亏损 |

**数据获取**：`westock quote` → `pe_fwd`；`consensus` → 2028年 `pe`

> **淘汰线**：<4分直接淘汰

**特别调整**：正常情况下，处于🔴高热状态的方向已被 Step 2 生命周期判断拦截，不会进入 Step 4。但对于"含金量高且仅做短线"的场景，统一**从维度④的原始得分中扣减2分**作为估值透支惩罚（维度④最低2分）。

#### 维度⑤：主题纯度（★★★）
| 分数 | 标准 |
|:---|:---|
| 7-10分 | 主题收入占比>30%，增速>30% |
| 4-6分 | 主题收入占比15-30%，是增量 |
| 1-3分 | 主题收入占比<15%，本质不相关 |

**数据获取**：`neodata` 查询主营收入构成 + 前5大客户

> **淘汰线**：<4分直接淘汰

#### 维度⑥：现金流验证（★★★）
| 分数 | 标准 |
|:---|:---|
| 7-10分 | 经营现金流/净利润>80% |
| 4-6分 | 50-80% |
| 1-3分 | <50% |

**数据获取**：`westock finance --type xjll` + `westock finance --type lrb`

> **淘汰线**：<4分直接淘汰

### 4.3 综合评级

```
加权总分 = 维度①×4 + 维度②×4 + 维度③×4 + 维度④×4 + 维度⑤×3 + 维度⑥×3
满分 = 190分
```

| 加权总分 | 评级 | 含义 | 操作建议 |
|:---:|:---|:---|:---|
| ≥140 | 🔴 强烈推荐 | 六维全部优秀，安全边际极高 | 核心仓位 |
| 110-139 | 🟡 值得关注 | 大部分维度优秀，个别需容忍 | 观察仓位 |
| 75-109 | 🟢 观察等待 | 有逻辑但有明显短板，等更好价格 | 等待回调 |
| <75 | ⛔ 淘汰 | 不满足基本投资要求 | 不参与 |

**硬性门槛**：任一维度低于淘汰线 → 综合评级最高为⛔淘汰。

## 工作规范和协议

### 数据协议
- 每个数字必须标注数据来源：[`quote`] / [`consensus`] / [`finance`] / [`neodata`] / [`report`] / [`WebSearch`]
- 无数据时标注 `[缺失]` 或 `[估算]`，绝不编造
- 数据超过90天标注 `[STALE]`
- 多个数据源冲突时，优先采信一手来源（交易所/公司公告/年报）

### 路由优先级
1. **neodata-financial-search**：默认优先，自然语言覆盖广
2. **westock-data**：需结构化数据/批量对比时切换
3. **WebSearch**：前两者均不满足时使用

### 合规底线
- 涉及内幕消息、未公开信息时明确拒绝
- 不评价尚未发布的政策或产品
- 不提供个股短期买卖建议
- 每份报告末尾必须标注免责声明

### 不构成投资建议
> 本报告仅供参考，不构成个人投资建议。
