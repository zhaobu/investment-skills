# 排除分析输出模板（Step 5）

> 对同一子板块中**进入初选但未最终入选**的标的，逐一说明排除原因。

## 标准化排除原因分类

| 排除原因 | 典型表述 |
|:---|:---|
| **龙头地位弱** | "虽有一定壁垒，但 XX 在行业内仅排第 X，龙头地位不稳固" |
| **估值透支** | "PE_fwd=XX，远高于行业均值，市场已被充分定价" |
| **主题纯度低** | "AI/主题相关收入占比仅 XX%，核心仍是传统业务" |
| **现金流差** | "经营现金流/净利仅 XX%，利润含金量不足" |
| **增速确定性弱** | "未来 2 年增速 CAGR 仅 XX%，缺乏明确增长驱动" |
| **技术路线风险** | "押注单一技术路线，存在被替代风险" |
| **客户集中度高** | "前 3 大客户占 XX%，单一客户依赖风险过高" |
| **情绪过热** | "雪球/股吧讨论量爆表，研报目标价集中度 >80%，情绪过热风险" |
| **纸面利润** | "3 年 OCF/NI 持续 < 50%，会计利润无现金支撑" |
| **强周期陷阱** | "PE 极低但行业处于盈利高点，下行风险大" |

## 输出格式

```markdown
#### ❌ [股票名称](代码) — 排除

**6 维总分**：XX/190 分

**核心短板**：[最关键的 1-2 个维度失分点]

**排除原因详述**：[200 字以内，对照入选标的的关键差距]

**重新考虑条件**：[什么情况下可以重新进入关注]
```

## 范例

```markdown
#### ❌ 宁德时代 (300750) — 排除

**6 维总分**：125/190 分

**核心短板**：维度④估值透支（PE_fwd=85x）

**排除原因详述**：
虽在维度①②③⑤⑥ 均表现优秀（稀缺性 9 分/龙头 10 分/确定性 9 分/纯度 9 分/现金流 8 分），
但当前 PE_fwd=85x 处于历史 90% 分位以上，2028E PE 仍有 50x。
考虑到锂电池行业未来 2 年增速可能从 50% 放缓至 30%，当前估值透支了至少 2-3 年增长。

**重新考虑条件**：
- PE_fwd 回落至 50x 以下（预计调整空间 -40%）
- 或储能业务超预期带动 2027E 净利上修 30%+
```

## 排除分析的原则

1. **必须有数据支撑**：每个排除原因都对应具体数字，不能用"我觉得"
2. **必须对照入选标的**：说明相对入选标的差距在哪
3. **必须给出重新考虑条件**：避免"一棒子打死"
4. **避免情绪化用语**：不用"垃圾股/毫无价值"等情绪词汇
