# 产业链拆解模板（Step 3）

> 本文件定义 7 种产业链类型的标准拆解模板（v1.1 从 4 种扩展为 7 种）。

---

## A 型：制造业链

**适用方向**：储能、新能源车、光伏、PCB、化工、机械、钢铁

**拆解模板**：
```
上游（原材料/设备）
  ↓
中游（制造/加工）
  ↓
下游（终端应用）
```

**实战案例**：
- **锂电池**：锂矿 → 正极/负极/电解液/隔膜 → 电芯 → 电池包 → 整车
- **光伏**：硅料 → 硅片 → 电池片 → 组件 → 电站
- **新能源车**：锂/钴/镍 → 电池/电机/电控 → 整车厂 → 充电桩/运营

**关键问题**：
- 上游：谁掌握资源？定价权在谁？
- 中游：技术壁垒在哪一环？（技术/规模/认证）
- 下游：谁是品牌方？谁掌握渠道？

---

## B 型：TMT 链

**适用方向**：AI、半导体、软件、5G、光通信、消费电子

**拆解模板**：
```
上游（底层硬件/IP）
  ↓
中游（芯片/模组/平台）
  ↓
下游（应用/服务）
```

**实战案例**：
- **AI**：算力芯片（GPU/ASIC）→ 服务器/光模块 → 大模型层 → 应用层（行业模型/Agent）
- **半导体**：硅片/光刻胶/设备 → IC 设计/制造/封测 → 终端芯片（消费/汽车/工业）
- **5G**：光器件/射频 → 基站/通信设备 → 运营商/应用

**关键问题**：
- 上游：是否被海外卡脖子？国产替代空间？
- 中游：技术迭代周期多长？（摩尔定律/制程升级）
- 下游：谁是杀手级应用？需求弹性如何？

---

## C 型：消费链

**适用方向**：白酒、医美、食品、家电、纺织服装

**拆解模板**：
```
上游（原料/包材）
  ↓
中游（品牌/生产）
  ↓
下游（渠道/零售）
```

**实战案例**：
- **白酒**：高粱/小麦/包材 → 酒企（茅台/五粮液/汾酒）→ 经销商 → 终端（商超/餐饮/电商）
- **医美**：玻尿酸/胶原蛋白原料 → 医美产品/器械 → 医美机构 → 消费者
- **家电**：钢/铝/铜 → 整机厂（美的/格力/海尔）→ 经销商/电商 → 终端用户

**关键问题**：
- 上游：成本占比多少？价格波动大不大？
- 中游：品牌力有多强？毛利率多高？
- 下游：渠道效率如何？线上线下比例？

---

## D 型：服务链

**适用方向**：金融科技、医疗、教育、SaaS、企业服务

**拆解模板**：
```
上游（基础设施）
  ↓
中游（平台/服务商）
  ↓
下游（终端用户）
```

**实战案例**：
- **AI 医疗**：医疗数据/算力 → AI 诊断平台 → 医院/患者
- **金融科技**：支付通道/数据源 → 金融科技平台 → 银行/消费者
- **SaaS**：云基础设施（IaaS）→ SaaS 平台 → 企业用户

**关键问题**：
- 上游：基础设施是否可获取？成本结构？
- 中游：网络效应有多强？用户切换成本？
- 下游：付费意愿？续费率？

---

## E 型：政策驱动链（v1.1 新增）

**适用方向**：碳交易、数据要素、电力改革、稀土、数字货币

**拆解模板**：
```
政策发布
  ↓
试点/示范
  ↓
规模化推广
  ↓
商业化变现
```

**实战案例**：
- **数据要素**：政策（《数据二十条》）→ 数据交易所（北京/上海/深圳）→ 数据应用方
- **碳交易**：政策（双碳目标）→ 试点（7 大试点）→ 全国碳市场 → 控排企业
- **数字货币**：政策（央行数字人民币）→ 试点（10+城市）→ 全面推广 → 商业应用

**关键问题**：
- 政策落地节奏？时间表明确吗？
- 哪些环节直接受益？哪些是间接受益？
- 商业化路径清晰吗？盈利模式是什么？

---

## F 型：周期资源链（v1.1 新增）

**适用方向**：有色金属、煤炭、化工、油气、钢铁

**拆解模板**：
```
上游资源（海外定价为主）
  ↓
中游冶炼/加工
  ↓
下游材料/应用
```

**实战案例**：
- **锂**：海外锂矿（澳洲/南美/非洲）→ 碳酸锂/氢氧化锂 → 电池正极
- **稀土**：国内矿/海外矿 → 稀土永磁材料 → 新能源车/风电
- **煤炭**：煤炭开采 → 煤化工/电力 → 钢铁/化工/电力

**关键问题**：
- 上游：资源储量？开采成本？海外依存度？
- 中游：技术壁垒？产能扩张周期？
- 下游：价格周期位置？需求弹性？

**特别提醒**：周期股估值方法与成长股不同（看 PB/ROE 而非 PE），6 维评分需重点关注维度③确定性。

---

## G 型：出海链（v1.1 新增）

**适用方向**：跨境电商、出口制造、新能源车出海、消费电子

**拆解模板**：
```
产能（国内）
  ↓
品牌/渠道（海外）
  ↓
终端用户（海外）
```

**实战案例**：
- **新能源车出海**：国内产能（宁德/比亚迪）→ 海外渠道（欧洲/东南亚/拉美）→ 海外用户
- **跨境电商**：国内供应链（Shein/Temu）→ 海外平台/物流 → 欧美消费者
- **家电出口**：国内产能（美的/海尔）→ 海外品牌（收购/自有）→ 海外用户

**关键问题**：
- 上游：成本优势是否可持续？
- 中游：渠道/品牌建设进度？关税壁垒？
- 下游：海外需求弹性？汇率风险？

**特别提醒**：出海标的需额外考虑：
1. 关税政策变化（中美/中欧贸易摩擦）
2. 汇率波动（人民币升值侵蚀利润）
3. 地缘政治风险

---

## 选择模板规则

1. 根据方向所属行业，选择对应的模板
2. 如果方向横跨多个类型（如"储能"既有 A 制造业属性也有 F 周期属性），以**价值量占比最高的环节为准**
3. 对于新兴/跨界热点方向，使用最接近的模板类型，并根据实际情况调整子板块划分

## 优先级排序标准

对拆解后的所有子板块，按以下标准决定**分析优先级**：

1. **价值量高**的子板块优先（毛利率高、不可替代性强）
2. **国产替代空间大**的子板块优先（技术壁垒高、国内企业少）
3. **直接受益 AI/数字化转型**的子板块优先（增量需求明确）
4. **周期底部反转**的子板块优先（仅对 F 型周期资源链）
5. **政策直接受益**的子板块优先（仅对 E 型政策驱动链）

## 子板块信息清单

每个子板块需列出：
- 子板块名称
- 核心业务描述（一句话）
- 在该方向中的价值量占比（估算）
- 该子板块的竞争格局（集中/分散/寡头）
- A 股主要上市公司清单（代码+名称）

> **数据获取方式**：`neodata` 查询产业链结构 + `westock sector --search` 找板块成分股 + WebSearch 补充
