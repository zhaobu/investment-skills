# 市场环境判断细则（Step 0）

> 本文件定义市场环境四象限的判断标准和边界情况处理。

## 四象限判断标准

### 🟢 强势市场

| 指标 | 阈值 |
|:---|:---|
| 指数走势 | 5 日均线 > 10 日均线 > 20 日均线（多头排列） |
| 涨跌比 | > 1.5（涨家数/跌家数） |
| 成交额 | > 8000 亿（沪深两市合计） |
| 情绪 | 连板高度 > 5，涨停家数 > 60 |
| 板块效应 | 有持续 3+ 日的领涨主线 |

**策略**：进攻（重仓做多），仓位 7-9 成。

### 🟡 震荡市场

| 指标 | 阈值 |
|:---|:---|
| 指数走势 | 均线纠缠，无明显趋势 |
| 涨跌比 | 1:1 左右 |
| 成交额 | 5000-8000 亿 |
| 情绪 | 连板高度 3-5，涨停家数 30-60 |
| 板块效应 | 轮动为主，无持续主线 |

**策略**：精选（聚焦核心方向），仓位 4-6 成。

### 🔴 弱势市场

| 指标 | 阈值 |
|:---|:---|
| 指数走势 | 5 日均线 < 10 日均线 < 20 日均线（空头排列） |
| 涨跌比 | < 0.7 |
| 成交额 | < 5000 亿 |
| 情绪 | 连板高度 < 3，涨停家数 < 30，跌停家数 > 涨停 |
| 板块效应 | 无主线，前期热点快速轮动后熄火 |

**策略**：控制（轻仓或空仓），仓位 0-3 成。

### ⚫ 退潮市场

| 指标 | 阈值 |
|:---|:---|
| 指数走势 | 连续 3 日下跌 + 跌破关键支撑位 |
| 涨跌比 | < 0.5 |
| 成交额 | 连续萎缩（< 4000 亿） |
| 情绪 | 跌停家数 > 50，连板高度归零 |
| 系统性风险 | 重大利空（美联储加息/地缘冲突/汇率暴跌） |

**策略**：观望（不操作），仓位 0 成。

## 数据获取

```bash
# 指数走势
neodata "上证指数 2026年 走势 均线"
westock-data kline sh000001 --period day --limit 20

# 成交额
neodata "A股 当日成交额"

# 涨跌家数
neodata "A股 涨跌家数 今日"

# 情绪（涨停数/连板高度）
westock-data hot stock
```

## 边界情况处理

| 特殊情况 | 处理规则 |
|:---|:---|
| 弱市但方向是防御性板块（医药/必选消费/公用事业） | 允许做 Step 3-6，但仓位上限 3 成 |
| 弱市但有政策强行催化（突发重磅政策） | 允许做 Step 3-6，标注"环境风险"，仓位上限 5 成 |
| 震荡市但多个方向同时触发 | 选持续性最好的 1 个方向深挖，其余进监控名单 |
| 方向横跨上下游（"AI+医药"） | 按主要收入来源（>60%）归入对应产业链模板 |
| 同板块多只标的评分接近（<10 分差距） | 优先 PE 更低，其次 OCF/NI 更高 |

## 关键规则

> **在🔴弱势或⚫退潮市场下，默认跳过产业链深挖（Step 3-6）**。
> 例外仅限防御性板块或政策强行催化，且需降低仓位。
