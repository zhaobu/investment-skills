# 实战场景模板

## 场景 1：周复盘（每周固定时间执行）

**输入**：持仓列表
**输出**：周复盘报告

**流程**：
```
1. 列出所有持仓标的
2. 对每只标的快速检查：
   - 监控指标是否有突破阈值
   - 是否有重大新闻/公告
   - 价格是否触发加减仓信号
3. 生成周复盘表
4. 如有信号触发，生成决策建议
5. 输出到 /持仓周报-[YYYY-MM-DD].md
```

**输出模板**：

```markdown
# 📊 持仓周报 — [YYYY-MM-DD]

## 整体表现
- 组合周涨跌：[+/-X%]
- 沪深 300 周涨跌：[+/-X%]
- 超额收益：[+/-X%]
- 当前仓位：[X%]
- 现金比例：[X%]

## 各标的表现

| 代码 | 名称 | 仓位 | 周涨跌 | 关键事件 | 操作 |
|:---:|:---|:---:|:---:|:---|:---:|
| | | | | | |

## 信号触发

### 加仓信号
- [ ] [标的] - [信号类型] - [建议加仓 X%]

### 减仓信号
- [ ] [标的] - [信号类型] - [建议减仓 X%]

## 下周关注
- [事件 1]
- [事件 2]
```

## 场景 2：单标的全周期跟踪

**输入**：建仓时的买入记录
**输出**：从建仓到退出的完整跟踪

**流程**：
```
1. 读取建仓档案
2. 拉取最新监控指标
3. 对照建仓时的核心假设
4. 输出当前逻辑兑现度评估
5. 生成加/减/清仓建议
```

**输出模板**：

## 场景 3：组合体检（每月一次）

**输入**：当前持仓组合
**输出**：组合整体评估报告

**流程**：
```
1. 计算组合整体表现
2. 评估各标的的相关性（避免过度集中）
3. 识别需要重新评估的标的
4. 检查是否有标的触发减仓/清仓信号
5. 输出调仓建议（v1.1 新增）
```
