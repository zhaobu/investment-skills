## 兑现度判定标准（v1.1 加权版）

| 兑现度 | 加权兑现度 | 操作建议 |
|:---|:---:|:---|
| **A. 完全兑现** | ≥ 80% | 继续持有，坚定信心 |
| **B. 部分兑现** | 60-79% | 继续持有，但密切关注 |
| **C. 不及预期** | 40-59% | 减仓至观察仓位 |
| **D. 被推翻** | < 40% | 清仓 |

## 量化对比：v1.0 vs v1.1

**示例场景**：3 核心指标 + 2 辅助指标 = 5 个指标

| 情境 | v1.0 判定 | v1.1 判定 | 差异 |
|:---|:---:|:---:|:---|
| 2 核心看多 + 1 辅助看多 | 3/5 = 60% → B | (2×2+1×1)/(3×2+2×1) = 5/8 = 62.5% → B | 一致 |
| 1 核心看多 + 2 辅助看多 | 3/5 = 60% → B | (1×2+2×1)/(3×2+2×1) = 4/8 = 50% → C | **v1.1 更严** |
| 0 核心看多 + 3 辅助看多 | 3/5 = 60% → B | (0×2+3×1)/(3×2+2×1) = 3/8 = 37.5% → D | **v1.1 严格区分** |

**v1.1 优势**：核心指标破 1 个立即反映在分母里，不会被辅助指标掩盖。
