# 📊 投资决策 Skill 体系 · WorkBuddy

> **一个完整的 A 股投资决策工具链：从发现机会、评估标的到管理持仓，全部覆盖。**

---

## 🎯 这是什么

这是一套运行在 [WorkBuddy](https://workbuddy.app) / [Cursor](https://cursor.sh) 等 AI Editor 中的 **投资决策 Skill** 集合。由 5 个相互配合的 Skill 组成，覆盖 A 股投资的完整闭环：

```
市场热点扫描 → 产业链深挖 → 板块精选 → 六维评分 → 持仓管理 → 退出复盘
```

**核心原则**：不追涨杀跌，不做情绪化交易。研究结论要经得起 3 年后回看。

## 🚀 快速上手

### 前置条件

这些 Skill 依赖以下金融数据源：

- **neodata-financial-search**：自然语言金融数据搜索（A 股/港股/美股行情、财报、板块、宏观等）
- **westock-data**：腾讯自选股结构化行情数据（实时行情、资金流向、机构评级、一致预期等）

Skill 会自动按优先级（neodata → westock-data → WebSearch）路由数据请求。

### 安装

```bash
# 克隆到本地
git clone https://github.com/zhaobu/investment-skills.git

# 将 Skill 复制到 WorkBuddy 的 skills 目录
cp -r investment-skills/* ~/.workbuddy/skills/
```

### 一句话开始

打开 AI Editor，直接输入：

> **"我想买股票但不知道买什么"**

系统会自动判断当前阶段，调用对应的 Skill 完成分析。

或者试试这些：

|:---|:---|
| 不知道市场在炒什么 | "帮我跑一下当前热点深挖" |

## 📋 Skill 详解

### 🔥 hotspot-chain-hunter — 热点产业链深挖选股器

**流程**：

```
Step 0: 市场环境判断（强市/震荡/弱市/退潮）
Step 1: 市场热点扫描（Top 5 方向）
Step 1.5: 热点持续性验证（排除一日游假热点）
Step 2: 含金量评估 + 生命周期位置（萌芽/主升/高潮/退潮）
Step 3: 产业链拆解（4 种模板：制造业/TMT/消费/服务）
Step 4: 六维评分（每个子板块精选 Top 1-2）
Step 5: 排除分析
Step 6: 输出完整报告
```

**特色机制**：
- 🛡️ 持续性四维验证：成交额/中军股/催化验证/资金无出逃
- 📍 生命周期位置判断：萌芽重仓、主升持有、高潮不追、退潮清仓
- 🔍 方向质量预检：避免"好方向但 A 股没好标的"
- 🕵️ 潜伏池 + 轮动触发条件

---

### 🎯 sector-stock-hunter — 板块精选7步法选股器

**解决什么问题**：你知道某个板块有行情（如"半导体材料涨了"），但板块内有几十只股票不知道买谁。本 Skill 将板块内的全部成分股通过7步筛选，最终只输出 1-2 只最适合1-3年持有的标的。

**7步流程**：

```
Step 0: 数据准备 → 板块代码 + 成分股 + 研报
Step 1: 热度初判 → YTD涨幅/PE分位/资金流向交叉验证
Step 2: 驱动拆解 → 上涨原因 → 3年可持续性判断
Step 2.5: 板块放弃 → 无持续因素→直接放弃（流程终止）
Step 3: 排雷预筛选 → 6类淘汰（主题不符/泡沫/亏损/纸面利润...）
Step 4: 六维评分 → 幸存标的逐一打分（复用 six-dimension-hunter 框架）
Step 5: 催化剂日历 → 未来3-6个月关键事件
Step 6: 最终精选 → 1-3年持有加权矩阵决出 Top 1-2
Step 7: 投资逻辑记录 → thesis-tracker 存档
```

**关键关卡（Step 2.5）**：
- 如果板块上涨驱动因素中没有一个是可持续3年以上的 → **流程立即终止**，直接告诉用户"这个板块不适合1-3年持有"
- 不浪费后续排序和评分资源

**加权矩阵（Step 6）**：

| 维度 | 权重 | 评分逻辑 |
|:---|:---:|:---|
| 2028E PE | 30% | 远期估值越低越好 |
| 现金流质量 | 20% | 经营现金流/净利润比率 |
| 护城河/稀缺性 | 20% | 竞争优势持久性 |
| 3年净利CAGR | 15% | 增长动力强度 |
| 主题纯度 | 10% | 赛道纯粹度 |
| 当前安全边际 | 5% | 回调空间 |

**特色**：
- 🛡️ **前置板块放弃机制**：避免在无持续性的板块上浪费分析精力
- 📐 **跨标的加权矩阵**：用6维权重从多只高分标的里再精选1只
- 🔄 **与6个Skill联动**：输入→sector-stock-hunter→six-dimension-hunter→thesis-tracker

---

**六维评分框架**（满分 190 分）：

| 维度 | 权重 | 核心标准 |
|:---:|:---:|:---|
| ①稀缺性 | ★★★★ | 全球/国内竞争者数量、技术壁垒 |
| ②龙头地位 | ★★★★ | 全球/国内市占率排名 |
| ③3-5年确定性 | ★★★★ | 下游需求确定性、净利 CAGR |
| ④估值未透支 | ★★★★ | PE_fwd、2028E PE 消化程度 |
| ⑤主题纯度 | ★★★ | 主题相关收入占比及增速 |
| ⑥现金流验证 | ★★★ | 经营现金流/净利润比率 |

**评级**：
- 🔴 ≥140 分：强烈推荐（核心仓位）
- 🟡 110-139 分：值得关注（观察仓位）
- 🟢 75-109 分：观察等待（等回调）
- ⛔ <75 分：淘汰（不参与）

**特色机制**：
- 🎯 赛道差异化门槛（AI 主题纯度 vs 白酒赛道不同标准）
- 🔍 5 类预期差识别（估值错配/增速错配/认知错配/财务质量错配/减持噪音错配）
- 📐 同分去重规则

### 📋 thesis-tracker — 投资逻辑跟踪器

**流程**：

```
Step 0: 持仓档案初始化
Step 1: 买入逻辑记录（必须可证伪）
Step 2: 监控指标设定（可量化+可获取+有时效+有阈值）
Step 3: 定期复盘（周/月/季）
Step 4: 信号触发识别（5 类加仓 + 6 类减仓 + 4 类清仓）
Step 5: 加减仓决策执行
Step 6: 退出复盘总结
```

**特色机制**：
- ⚡ 3 层止损防线：初始止损（-15%）→ 跟踪止损 → 硬止损
- 🦢 黑天鹅应急处理：3 级响应机制（严重/中等/轻微）
- 🧠 7 种常见决策错误对照表（沉没成本/确认偏误/锚定效应等）
- 🔄 退出强制复盘，经验反哺评分模型

---

### 🧭 invester-dp — 三步投资法决策导航

**核心作用**：判断你当前处于哪个阶段，告诉你该调用哪个 Skill。

4 大场景入口：
- 🅰 还没买，但知道板块 → 引导到 sector-stock-hunter → six-dimension-hunter
- 🅰 还没买，不知道买什么 → 引导到 hotspot-chain-hunter → sector-stock-hunter → six-dimension-hunter
- 🅱 已经买了 → 引导到 thesis-tracker
- 🅲 已经退出 → 引导到 thesis-tracker Step 6（复盘）
- 🅳 组合体检 → 引导到 thesis-tracker 场景 3
