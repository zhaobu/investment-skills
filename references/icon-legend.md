## 9. 信号触发

> **注意**：本节图标与第 1 节（评级）使用同一套颜色，但含义不同。信号触发用图标 + 文字双标签避免歧义。

| 图标 | 文字 | 信号类型 |
|:---:|:---|:---|
| ➕ | **加仓** | 加仓信号 |
| ➖ | **减仓** | 减仓信号 |
| 🚪 | **清仓** | 清仓信号 |
| ⏸️ | **持有** | 持有观察 |
| 🔄 | **调仓** | 调仓信号 |

> **报告中请使用 `➕ 加仓` / `➖ 减仓` / `🚪 清仓` 格式**（图标+文字），避免单独使用 🔴🟡🟢 颜色标识。

## 使用规则

1. **同一概念全 Skill 统一**：如"加仓"在 hotspot-chain-hunter / sector-stock-hunter / six-dimension-hunter / thesis-tracker 中都用 ➕
2. **颜色优先于图标**：可读性第一，色盲友好
3. **避免图标滥用**：每屏不超过 8 种不同图标
4. **首次出现给说明**：报告中首次使用某图标时在图例中说明
